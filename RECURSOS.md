# 📚 Recursos Adicionales para Aprender GitHub

Si quieres profundizar en el uso de **Git** y **GitHub**, aquí tienes una selección de los mejores materiales gratuitos para pasar de principiante a experto.

---

## 🌐 Documentación Oficial
* **[Git Documentation](https://git-scm.com/doc):** El manual definitivo. Incluye el libro "Pro Git" completo y gratuito.
* **[GitHub Docs](https://docs.github.com/es):** La guía oficial paso a paso para todas las funciones de la plataforma.

## 🎓 Cursos e Interactivos
* **[GitHub Skills](https://skills.github.com/):** Aprende usando GitHub. Son repositorios interactivos que te enseñan a hacer Merges, resolver conflictos y más.
* **[Learn Git Branching](https://learngitbranching.js.org/?locale=es_ES):** La herramienta visual más famosa para entender cómo funcionan las ramas (branches) mediante un juego.

## 🛠️ Herramientas Recomendadas
* **[Visual Studio Code](https://code.visualstudio.com/):** Incluye una barra lateral de "Source Control" que facilita mucho el trabajo visual con Git.
* **[GitHub Desktop](https://desktop.github.com/):** Si prefieres una interfaz visual sencilla en lugar de la terminal.

---
> **Tip:** No intentes memorizar todos los comandos. Lo más importante es entender el flujo: `add` -> `commit` -> `push`.
